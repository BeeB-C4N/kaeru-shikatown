  • GitHub Pages を使ってWebアプリを世界に公開する

Step 1：GitHub Pagesでホスティング

今回作成するマップアプリは、各自のGitHubアカウントを使って「GitHub Pages」というサービスでWeb上に公開（ホスティング）します。
これにより、URLさえ知っていれば誰でもあなたのアプリにアクセスできるようになります。

⚠️ セキュリティに関する超重要事項

GitHub Pagesは「静的サイトを無料でホスティングできる」強力なツールですが、リポジトリ（ソースコード）は基本的に全世界に公開されます。

  • 本名、電話番号、非公開の業務データ（個人情報）を含めないこと
  • APIキーやパスワードをコード内に直接書き込まないこと

これ以降のスライドで紹介するコードは、コピペして index.html として保存し、GitHubにプッシュして動作を確認しましょう。

Step 2：Leaflet入門（基本機能）

Leaflet（リーフレット） は、JavaScriptで記述できる軽量で人気のある地図ライブラリです。
背景の地図データには、誰でも自由に編集・利用できる「地図のWikipedia」こと OpenStreetMap を使用します。

以下のコードは、マーカー、ポップアップ、ライン、カスタムアイコンに加えて、レイヤの切り替え機能（Layer Control）など、基本的な機能を網羅したテンプレートです。

index.html (Leaflet基本) 
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Leaflet入門マップ</title>
    <!-- LeafletのCSSを読み込み -->
    <link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css" />
    <style>
        body { margin: 0; padding: 0; }
        #map { width: 100vw; height: 100vh; }
    </style>
</head>
<body>
    <div id="map"></div>

    <!-- LeafletのJSを読み込み -->
    <script src="https://unpkg.com/leaflet@1.9.4/dist/leaflet.js"></script>
    <script>
        // 1. 地図の初期化
        var map = L.map('map').setView([37.004365, 136.775708], 11);

        // 2. 背景地図（OpenStreetMap）の追加
        var osm = L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
            attribution: '&copy; <a href="https://www.openstreetmap.org/copyright">OpenStreetMap</a> contributors'
        }).addTo(map);

        // 3. マーカーとポップアップの作成（後でレイヤにまとめるため変数に入れる）
        var normalMarker = L.marker([37.004365, 136.775708])
            .bindPopup('<b>高浜バスターミナル</b><br>志賀町の中心地付近です。');

        // 4. カスタムアイコンを使ったマーカー
        var redIcon = L.icon({
            iconUrl: 'https://raw.githubusercontent.com/pointhi/leaflet-color-markers/master/img/marker-icon-red.png',
            shadowUrl: 'https://cdnjs.cloudflare.com/ajax/libs/leaflet/0.7.7/images/marker-shadow.png',
            iconSize: [25, 41], iconAnchor: [12, 41], popupAnchor: [1, -34]
        });
        var customMarker = L.marker([37.140558, 136.725355], {icon: redIcon})
            .bindPopup('道の駅とぎ海街道（カスタムアイコン）');

        // 5. ライン（線）を引く
        var latlngs = [
            [37.004365, 136.775708],
            [37.140558, 136.725355]
        ];
        var routeLine = L.polyline(latlngs, {color: 'red', weight: 4, opacity: 0.7});

        // 6. 複数の要素を「レイヤグループ」にまとめる
        var myFeatures = L.layerGroup([normalMarker, customMarker, routeLine]).addTo(map);

        // 7. レイヤコントロール（右上の切り替えスイッチ）を追加
        var baseMaps = {
            "基本マップ (OSM)": osm
        };
        var overlayMaps = {
            "表示アイテム": myFeatures
        };
        L.control.layers(baseMaps, overlayMaps).addTo(map);

        // 初期状態でポップアップを開く
        normalMarker.openPopup();
    </script>
</body>
</html>

Step 3：オープンデータの準備

地図上に大量のピンを立てるため、今回は志賀町のオープンデータカタログにある「公衆トイレ情報」を使用します。
以下のボタンからCSVファイルをダウンロードし、index.html と同じフォルダに保存してください。

志賀町 公衆トイレ情報 (CSV)

ファイル名：toilet_shikatown.csv

ディレクトリ構成の確認： GitHubにプッシュする際も、HTMLファイルとCSVファイルが同じ階層にあることを確認してください。

Step 4：CSVデータを地図にプロットする

JavaScriptの fetch 関数を使って、同じフォルダにあるCSVを読み込み、ループ処理で全てのトイレを地図上にプロットします。
ポップアップには、男性・女性トイレの数も表示させています。

index.html (CSV読み込み版) 
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>志賀町トイレマップ</title>
    <link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css" />
    <style>
        body { margin: 0; padding: 0; }
        #map { width: 100vw; height: 100vh; }
    </style>
</head>
<body>
    <div id="map"></div>

    <script src="https://unpkg.com/leaflet@1.9.4/dist/leaflet.js"></script>
    <script>
        var map = L.map('map').setView([37.08, 136.75], 11);
        L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
            attribution: '&copy; OpenStreetMap contributors'
        }).addTo(map);

        // 同じフォルダにあるCSVファイルを読み込む
        fetch('./toilet_shikatown.csv')
            .then(response => response.text())
            .then(text => {
                // 行ごとに分割
                const rows = text.split('\n');
                
                // 1行目はヘッダーなので、2行目(i=1)からループ処理
                for (let i = 1; i < rows.length; i++) {
                    if (!rows[i].trim()) continue; // 空行はスキップ
                    
                    const cols = rows[i].split(',');
                    const name = cols[0];
                    const lat = parseFloat(cols[5]); // 緯度
                    const lng = parseFloat(cols[6]); // 経度
                    
                    const maleCount = cols[7];
                    const femaleCount = cols[8];
                    const barrierFree = cols[9];
                    
                    // 緯度経度が正しく取得できた場合のみマーカーを設置
                    if (!isNaN(lat) && !isNaN(lng)) {
                        const popupHtml = `
                            <b style="color:#0B67B2; font-size:1.1em;">${name}</b><br>
                            <hr style="margin:5px 0;">
                            男性用: ${maleCount} / 女性用: ${femaleCount}<br>
                            バリアフリー: ${barrierFree}
                        `;
                        L.marker([lat, lng]).addTo(map).bindPopup(popupHtml);
                    }
                }
            })
            .catch(error => console.error('CSV読み込みエラー:', error));
    </script>
</body>
</html>

※実際のPC（ローカル環境）でHTMLをダブルクリックして開くと、セキュリティ制約（CORSエラー）により fetch がブロックされます。
GitHub Pagesにプッシュして、Web上のURLから動作を確認してください。（右のプレビュー枠はシミュレーション用に動作させています）

演習課題：オープンデータマップを作ろう

今回の知識を使って、あなたオリジナルのマップアプリを作成し、GitHub Pagesで公開してみましょう。

課題内容：AEDと観光施設の重ね合わせマップ

  1. 志賀町の「AED設置箇所一覧」と「観光施設一覧」の2つのCSVデータを読み込む。
  2. レイヤ機能： 2つのデータを別々のレイヤグループとして地図にプロットし、初期表示は両方オン、右上のコントロールからオン/オフを切り替えられるようにする。
  3. 応用（Googleマップ連携）： 各マーカーのポップアップ内に「ここに行く」ボタンを配置し、クリックするとGoogleマップの経路検索（ナビ）が別タブで開くようにする。
    ※ヒント: GoogleマップのURLは https://www.google.com/maps/dir/?api=1&destination=緯度,経度 を使用します。

AIへのプロンプト例：

以下のLeafletマップアプリのコードをベースに、新しい機能を追加してください。
1. 2つのCSV「aed_locations.csv」「sightseeing.csv」を読み込んでください。
2. それぞれ別のレイヤグループとして地図に追加し、L.control.layersを使ってチェックボックスで表示/非表示を切り替えられるようにしてください。初期状態は両方表示（オン）です。
3. 各マーカーのポップアップには、施設名の他に「ここに行く」というボタンを設置し、Googleマップの経路検索（destination=緯度,経度）が別タブで開くようにしてください。
