LGWAN環境 Webアプリ開発研修

AIとWebページを作ってみる

HTML・CSS・JavaScriptによる静的サイト構築の基礎

本日の目標

  • Webページが何から作られているかを知る（HTML/CSS/JS）
  • 生成AI（ChatGPTやGemini）を使った開発手法を体験する
  • LGWAN環境でも開発できる「ローカル環境構築」をマスターする

準備：AIとメモ帳で作るWebページ

LGWAN環境のようなインターネットから切り離された環境でも、Windows標準の「メモ帳」があればWeb開発が可能です。
この研修では、完全なコード（コピペしてそのまま動く形）を扱い、ブラウザで確認します。

プロトタイプ開発の基本フロー

  1. AIに指示を出す： インターネットが使える端末で、AIにWebページのコードを作成させます。
  2. コードを持ち込む： 生成されたコードをテキストデータとしてLGWAN環境に持ち込みます。
  3. メモ帳に貼り付ける： メモ帳を開き、コードをすべて貼り付けます。
  4. 拡張子を変えて保存： ファイル名を index.html にして保存します。
    ※保存時、文字コードが「UTF-8」になっていることを確認してください。
  5. ブラウザで開く： 保存したファイルをダブルクリックして、Edgeなどのブラウザで開きます。

重要ポイント： Webページとして正しく表示させるには、<html> や <body> などの「お作法（タグ）」が含まれた完全なコードである必要があります。以降のステップでは、すべて完全なコードを用意しています。

Step 1：HTMLだけでページを作る

HTML（HyperText Markup Language）は、Webページの「骨組み・構造」を作る言語です。

まずはデザイン（色やレイアウト）が一切ない、純粋な骨組みだけのページを見てみましょう。コードをコピーしてメモ帳に貼り付け、動作を確認してみてください。

index.html 
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>志賀町の紹介</title>
</head>
<body>
    <h1>志賀町の紹介</h1>
    <p>自然豊かな素晴らしい町です。</p>

    <h2>おすすめポイント</h2>
    <ul>
        <li>美しい海岸線（増穂浦海岸など）</li>
        <li>美味しい海産物</li>
        <li>歴史ある巌門</li>
    </ul>

    <button>詳細を見る</button>
</body>
</html>

Step 3：JavaScriptで動きをつける（機能紹介）

JavaScript（JS）は、Webページに「動き・機能」を追加する言語です。
AIに指示を出すためには、実現したい機能の「一般的な名称（専門用語）」を知っておくことが非常に重要です。

AIへの指示で役立つキーワード

  • ハンバーガーメニュー： スマホ等で三本線（≡）を押すと出てくるメニュー。
  • モーダルウィンドウ： 画面の真ん中にポンッと浮き出て、背景が暗くなる小窓。
  • アコーディオン： 見出しをクリックすると、ビヨーンと詳細が開閉する仕組み。
  • カルーセル（スライダー）： 画像や要素が横にスライドして切り替わる機能。
  • ドラッグ＆ドロップ： マウスで要素を掴んで、別の場所に移動させる操作。

次のページでは、これまで学んだ「志賀町の紹介」のコードをベースに、上記の機能をすべて盛り込んだ「完全版のコード」を見てみましょう。

Step 4：ディレクトリ構造と画像パス

ローカル環境で画像を表示するには、HTMLファイルと同じ場所（ディレクトリ）に画像ファイルを準備し、「相対パス」で指定する必要があります。

1. フォルダ構成を作る

プロジェクトフォルダ ... 任意の名前のフォルダ
 ├── index.html ... 作成したHTMLファイル
 └── images ... 新しく画像用のフォルダを作る
     └── bg.png ... ここに画像を配置する

2. HTMLで画像を読み込む

<img> タグの src 属性に、HTMLファイルから見た画像の位置（パス）を書きます。
./ は「今いるフォルダ」を意味します。

画像の読み込み例 (index.htmlの一部) 
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>画像の読み込み</title>
</head>
<body>
    <!-- imagesフォルダの中の bg.png を読み込む -->
    <img src="./images/bg.png" alt="タイトル画像" style="max-width: 100%;">

    <h1>志賀町の紹介</h1>
    <p>上に画像が表示されていれば、パスの指定は成功です！</p>
</body>
</html>

もし画像が表示されない場合は、「フォルダ名が間違っていないか」、「拡張子が .png や .jpg で合っているか」を確認しましょう。

演習課題：プロトタイプを作ってみよう

生成AI（ChatGPT等）を使って、「志賀町優良特産品」を紹介するWebサイトのプロトタイプを作成し、メモ帳を使ってローカルで表示してみましょう。

プロトタイプ作成のTips（LGWAN環境向け）

  • AIへの指示に「完全なHTMLファイルとして、CSSやJSも1つのファイルにまとめて出力して」と付け加えると、コピペが確実になります。
  • LGWAN環境などインターネットに繋がらない環境で動かすため、「外部のライブラリ（CDNなど）は参照せず、独自の実装で記述して」と指定するのがコツです。
